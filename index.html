設計の特徴

01

ロジック共有アーキテクチャ

管理台帳では Desktop UI と Web API が同一の services/ を参照。UI 変更と業務ルール変更を独立させ、二重実装を排除。

02

テスト駆動の品質担保

shaft_ring_ocr（36 テスト）、U-WAVE 分岐送信（40+ テスト）など、パケット解析・ルーティング・OCR パイプラインに pytest を適用。

03

配布まで含めた開発

PyInstaller EXE 化、WiX MSI、ランタイム DLL 除外（OS 同梱 DLL 利用）、共有フォルダ自動更新まで一貫対応。

04

現場機器との実接続

BLE・シリアル・MQTT・TCP/IP など、実機プロトコルに基づく連携。仮想 COM・エコーツールで開発・検証環境も整備。
